WYRDHORD - a collection of web-based sound instruments that explore touch, feedback, degradation, and emergence.

You may want to start by making some audio files available on your device. There is a link at the top of the menu page to various test audio, or you can go here: https://tinyurl.com/Wyrdaudio

The apps should work on all browser equipped devices, please send or link me to anything you create with them!
